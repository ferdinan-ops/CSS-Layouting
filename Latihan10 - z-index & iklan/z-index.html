Z-index CSS Layouting

z-index digunakan utk mengatur tumpukan dari elemen ketika elemennya non-static. z-index berguna utk menentukan elemen yg bertumpuk" memiliki elemen yg paling depan sesuai dengan yg diinginkan. z-index memiliki value/ nilai angka semakinbesar angkanya maka elemen yg memiliki nilai itulah yang berada di depan.

Untuk membuat elemen ingin berada di paling depan biasanya nilainya dibuat sperti ini

z-index: 9999;

Untuk membuat elemen ingin berada di paling belakang biasanya nilainya dibuat sperti ini

z-index: -9999;

Contoh Penggunaan z-index :

Pada kondisi dibawah ini kita dapat melihat bahwa kotak yg berada di paling depan adalah kotak biru, karena kotak biru memiliki z-index: 1; dan yg lainnya tdk memiliki nilai z-index.

Berikut contoh script CSS-nya


        div {
            width: 200px;
            height: 200px;
            position: absolute;
        }
        
        /* kotak terdepan */
        .satu {
            background-color: lightblue;
            z-index: 1;
        }
        
        .dua {
            background-color: lightgreen;
            top: 100px;
            left: 100px;
        }
        
        .tiga {
            background-color: salmon;
            top: 200px;
            left: 200px;
        }
        
        .pembungkus {
            width: 396px;
            height: 396px;
            border: 2px solid green;
            padding: 10px;
        }
