CSS Reset

CSS reset ini berguna untuk membuat nilai margin dan padding dari semua elemen tereset atau menjadi 0. Nah, hal ini membuat kita dapat mengendalikan penuh halaman kita. Ada banyak cara menggunakannya yaitu membuat tanda "*" di css dan membuat margin dan paddingnya 0. Tanda "*" berarti memilih semua elemen yang ada di html dan hal ini tidak disarankan karna tanda ini sangat berat karna membuat tag yg tdk terpakai pun ikut.

Nah, cara yang digunakan ialah mengunjungi website Erik Meyer CSS Reset dgn menyalin code website tsb dan menempelnya di file css membuat CSS dapat tereset.
