Penggunaan Border

    Border Struktur :

  • border-arah: width style color
  • Cth: border-top: 5px solid black

    Property Border :

  1. border-top
  2. border-bottom
  3. border-left
  4. border-right

    Shorthand :

  • border: width style color
  • Cth: border: 5px solid black

    Style yang ada di border :

  1. solid : garis kotak
  2. dotted : titik putus-putus
  3. dashed : garis putus-putus membentuk kotak
  4. double : 2 garis kotak

*note: border juga menambah ukuran dari box sama seperti padding
