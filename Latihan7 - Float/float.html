Float CSS Layouting

Float merupakan properti pada CSS utk mengatur posisi sebuah elemen. Sebuah elemen dapat dipaksa utk berada di sebelah kiri atau kanan dari parent/pembungkusnya dgn menggunakan properti ini.

Value float :

  1. none (default)
  2. left = membuat elemen berada di kiri
  3. right = membuat elemen berada di kanan
