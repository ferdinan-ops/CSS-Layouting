Dimensi & Overflow

Dimensi

Setiap tag/elemen HTML memiliki sebuah dimensi kotak atau sering disebut box model. Box model tsb bisa dimodifikasi dengan porperty dimensi. Dimensi berguna untuk mengatur panjang dan lebar box atau kotak dimensi pada setiap elemen HTML

Property Dimensi :

  • width : berguna utk mengatur lebar
  • height : berguna utk mengatur tinggi

Value width & height :

  • px : nilainya pasti
  • % : nilai relativ thdp halaman web
  • in, cm, mm, pt, pc : nilai satuan ukuran.

Overflow

overflow berguna untuk membuat dimensi sesuai dengan ukuran box dengan menambahkan fitur dari overflow tsb seperti scroll down, menghilangkan sebagian isi elemen, membuat scroll horizontal dll. Contoh penggunaan overlow ini sperti kita mempunyai sebuah box berisi banyak tulisan namun box tsb tidak cukup menampungnya dan tulisan keluar dari box. Nah, dengan overflow inilah hal itu bisa diatasi.

Value overflow :

  • visible : jika ada content yg keluar dari parentnya dia akan keluar. Semua elemen secara default memiliki value visible.
  • auto : jika ada konten yg keluar dari parentnya maka akan dibuat scrolldown disamping kanan nya.
  • hidden : jika ada konten yg keluar dari parenynya maka konten yg keluar itu akan disembunyikan.
  • scroll : jika ada konten yg keluar dari parentnya maka akan dibuatkan scorlldown dan scroll horizontal.
