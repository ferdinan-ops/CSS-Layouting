Clear CSS Layouting

Clear ini berfungsi untuk menghentikan/membersihkan float dan mengatasi masalah yang dibuat property float yaitu membuat elemen jadi tidak dibaca oleh grup elemen parentnya. Namun banyak juga cara untuk dapat mengatasi hal tsb tetapi hanya property clear ini yang lebih efektif untuk digunakan dan disarankan.

Value Clear :

  • left : beri nilai left jika nilai float = left
  • right : beri nilai right jika nilai float = right
  • both : beri nilai both jika ingin membersihkan float bernilai left dan right

Cara menghentikan float :

  • Menggunakan property overflow
  • Menggunakan tag div kosong
  • Menggunakan teknik micro clearfix

  3. Menggunakan teknik micro clearfix

    4. Perlu diperhatikan teknik micro clearfix ini adalah buatan seseorang yang bernama Nicolas Galagher. Untuk menggunakan cara ini Anda harus menyalin dan menempel script yang ada di website milik Nicolas Galagher. Klik disini untuk menggunakannya.

    Berikut code clearfix :

                    /**
 * For modern browsers
 * 1. The space content is one way to avoid an Opera bug when the
 *    contenteditable attribute is included anywhere else in the document.
 *    Otherwise it causes space to appear at the top and bottom of elements
 *    that are clearfixed.
 * 2. The use of `table` rather than `block` is only necessary if using
 *    `:before` to contain the top-margins of child elements.
 */
.cf:before,
.cf:after {
    content: " "; /* 1 */
    display: table; /* 2 */
}

.cf:after {
    clear: both;
}

/**
 * For IE 6/7 only
 * Include this rule to trigger hasLayout and contain floats.
 */
.cf {
    *zoom: 1;
}
